Часть 2: Реализация аутентификации через API
**Цель:** Обеспечить безопасную регистрацию, вход и выход пользователя с использованием Laravel Sanctum.

Выполнено:
- Установка и настройка Sanctum
- Создание API-эндпоинтов (/signup, /login, /logout, /user)
- Серверная валидация данных (Form Requests)
- Обработка ошибок валидации на фронтенде
- Сохранение и передача JWT-токена
- Автоматическая загрузка данных пользователя при наличии токена
- Улучшение дизайна интерфейса (современный стиль, адаптивность)
